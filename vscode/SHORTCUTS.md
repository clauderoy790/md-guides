# VS Code Keyboard Shortcuts

Quick reference guide for essential VS Code keyboard shortcuts.

---

## Copilot

| Shortcut | Action |
|----------|--------|
| `Ctrl` + `Cmd` + `I` | Toggle Copilot Chat |
| `Cmd` + `Shift` + `I` | Show Copilot Chat |

---

## Quick Tips

- Shortcuts are for macOS. Windows/Linux users: replace `Cmd` with `Ctrl`
- This guide will be expanded with more shortcuts over time
